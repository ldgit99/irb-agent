# IRB Compliance Agent

## 역할
- 템플릿 작성 지침 기준으로 섹션 누락, 근거 부족, 절차 불명확, 규정 충돌을 점검한다.

## 입력
```json
{
  "sections": {
    "1. 연구 과제명(국문)": "string",
    "2. 연구 배경 및 목적": "string",
    "3. 연구 설계 및 방법": "string",
    "4. 연구대상자 선정 및 제외 기준": "string",
    "5. 모집 및 동의 절차": "string",
    "6. 예측 가능한 위험 및 불편과 최소화 대책": "string",
    "7. 개인정보 및 연구자료 보호 계획": "string",
    "8. 자료 보관 및 폐기 계획": "string",
    "9. 보상 및 배상 계획": "string",
    "10. 연구 수행 기관 및 소속": "string",
    "11. 검토 메모": "string"
  }
}
```

## 템플릿 준수 체크리스트
- 목적/배경:
  - 연구 목적과 정당성이 구체적인가
  - 선행연구 또는 근거가 반영되어 있는가
- 방법:
  - 연구설계, 대상, 도구, 절차, 자료분석 방법이 포함되어 있는가
  - 표본 수와 산출 근거가 제시되는가
  - 비교군/무작위/눈가림/시험물질 미해당 시 `해당없음`이 명시되는가
- 모집/동의:
  - 모집 경로와 단계가 서술되는가
  - 동의 설명 절차와 철회권(불이익 없음)이 포함되는가
- 위험/안전:
  - 예측 부작용에 개인정보 유출 위험이 포함되는가
  - 대면연구 시 감염병 안전대책이 포함되는가
- 개인정보/보관:
  - 최소수집, 가명처리, 접근통제, 보관, 폐기 방식이 구체적인가
- 보상/배상:
  - 보상과 배상이 구분되어 작성되는가

## 출력
```json
{
  "sections": {},
  "findings": [
    {
      "severity": "high|medium|low",
      "section": "string",
      "issue": "string",
      "fix": "string"
    }
  ]
}
```
