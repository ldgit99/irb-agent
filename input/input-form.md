# IRB 입력 양식

아래 항목을 작성하면 IRB 계획서 자동 생성에 사용됩니다.

## 필수 입력

* 연구 주제: 바이브 코딩을 활용한 교육이 교사의 문제해결력에 미치는 영향
* 연구 방법: 10차시 수업, 문제해결력 사전-사후 검사
* 연구 시작일: 2026-01-01
* 연구 종료일: 2026-06-30
* 연구 대상 유형: 교사
* 연구 대상 인원: 50

## 추천 입력

* 연구 설계 유형: 양적
* 동의 방식: 서면 동의
* 자료 수집 도구: 설문지
* 연구 위험 수준: 최소위험

## 선택 입력

* 수행기관: 경북대학교
* 소속: 사범대학 정보컴퓨터교육과
* 민감정보 수집 여부: 아니오
* 대면 진행 여부: 예

## 연구책임자 정보

* 연구책임자 성명: 이동국
* 연구책임자 전화번호: 053-950-5845

## 작성 팁

* 연구 방법은 수업 절차, 자료수집, 분석 방법이 보이게 작성하세요.
* 시작일/종료일은 YYYY-MM-DD 형식 권장.
* 연구 대상 유형과 인원은 심의에서 자주 확인하는 핵심 항목입니다.
